PROGRESS

By now, we finished a simple renderer with resterization algorithm and voxel octree construction.
In the first week, we did more research for details from different materials, correct some wrong consept we have in mind, and set up basic environment with C++, OpenGL which could render triangles. Then we finished our own framework which could render the scene with material and support moving camera. After that, we implemented the rasterization algorithm and the basic sparse voxel octree hierarchy construction.

GOALS and DELIVERABLES

We are a little behind our plan right now because the some parts that are not described in the paper is more complicated than what we thought at first. We haven't started the voxel cone tracing right now. Never the less, we still achieved our goal to render a voxelized scene.

For next three weeks, we plan to implement a light kernel structure and voxel cone tracing in one and a half week. Then, we will debug, optimize, create a real-time interactive demo for final presentation. We may put dynamic lights optional right now.

We still believe we can have a real-time rendering application that supports limited global illumination on large scenes.
